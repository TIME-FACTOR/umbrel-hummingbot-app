Hummingbot is running

Hummingbot is a CLI app. There is no web UI. Use Terminal (or SSH) to attach to the container and run the bot.

Connect via Terminal

On your Umbrel device (or via SSH), run:

docker exec -it mystore-hummingbot_hummingbot_1 bash

If your app id is different (e.g. sparkles-hummingbot), use that prefix: sparkles-hummingbot_hummingbot_1. List containers to see the exact name:

docker ps -a | grep hummingbot

Then start Hummingbot inside the container (see the image docs for the exact command, often hummingbot or similar).

Gateway (DEX)

If you enabled the Gateway profile, the DEX middleware API is available on port 15888 (e.g. http://<umbrel-host>:15888). Use it from Hummingbot when connecting to DEXs.
