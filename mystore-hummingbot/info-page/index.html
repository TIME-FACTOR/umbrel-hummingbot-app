Hummingbot (developer version)

► Open the Hummingbot CLI in your browser
Open CLI (browser terminal)

Then run hummingbot in the terminal to start the bot.

This app runs Hummingbot from source so you can replace files for custom models. It's a CLI app—use the button above or Terminal/SSH to attach.

Custom models / code: The app uses the clone in the source folder. On first start the container clones the repo there. Replace any files you need in that folder (e.g. under source/hummingbot/), then restart the app so changes are applied.

Connect via Terminal (SSH)

On your Umbrel device (or via SSH):

docker exec -it mystore-hummingbot_hummingbot_1 bash

If your app id differs, list containers: docker ps -a | grep hummingbot. Then run hummingbot inside the container.

Where is the source?

On the Pi, app data is under /mnt/data/umbrel/app-data/mystore-hummingbot/. The source/ folder there is the Hummingbot repo (cloned on first run). Edit files there and restart the app.

Gateway (DEX)

If you enabled the Gateway profile, the API is on port 15888 (e.g. http://<umbrel-host>:15888).
